# GroupHug Frontend

## Development

Requirements: PHP 8.3 (CLI), [Composer](https://getcomposer.org/), GroupHug server running on `localhost:8787`.

```shell
$ cd frontend
$ composer install
$ composer serve
```

The frontend runs on `localhost:8080`.

## Production

Requirements: PHP 8.3 (FPM), Composer, Caddy server, GroupHug server.
